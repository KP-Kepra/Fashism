# FASHISM

Frontend repository for latest fashion trends based on pop-stars.
The fashion ranking is made in Caffe.

## Installation

*  Install yarn globally by `npm install yarn -g`
*  `yarn install`
*  `yarn start`
*  Open 'localhost:8080' to open the app
